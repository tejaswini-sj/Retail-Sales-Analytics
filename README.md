# Retail Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes retail sales data using Python, SQL, and Power BI to identify business insights. It includes data cleaning, KPI analysis, sales trend analysis, and dashboard visualization for better business decision-making.

---

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- Matplotlib
- SQL (MySQL)
- Power BI
- VS Code
- MySQL Workbench

---

## 📂 Dataset

- Retail Sales Dataset
- Customer Demographics
- Product Categories
- Sales Transactions

---

## 📊 Dashboard Features

- Total Revenue
- Total Quantity Sold
- Average Order Value
- Category-wise Sales Analysis
- Monthly Sales Trend
- Gender-wise Sales Distribution
- Interactive Dashboard
